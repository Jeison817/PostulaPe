# PostulaPe

Sistema web de gestión de convocatorias y postulaciones desarrollado con PHP y MySQL bajo arquitectura MVC.

## Descripción

PostulaPe es una aplicación web orientada a la gestión de procesos de convocatoria y postulaciones. El sistema permite administrar usuarios, convocatorias y postulantes mediante una interfaz organizada y funcional.

Este proyecto fue desarrollado como parte de prácticas de desarrollo web y gestión de sistemas de información.

## Funcionalidades

- Inicio de sesión y autenticación de usuarios
- Gestión de convocatorias
- Registro y administración de postulantes

## Instalación

### 1. Clonar repositorio

```bash
git clone https://github.com/Jeison817/PostulaPe.git
```

### 2. Importar base de datos

Importar el archivo `.sql` en MySQL mediante phpMyAdmin o MySQL Workbench.

### 3. Configurar conexión

Editar los parámetros de conexión a la base de datos según el entorno local.

### 4. Ejecutar proyecto

Abrir el proyecto en un servidor local como:

- XAMPP
- Laragon
- WAMP
